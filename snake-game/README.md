# Snake Game

This project is a simple implementation of the classic Snake game using Java Swing. The game is designed to be played in a graphical window.

## Project Structure

```
snake-game
├── src
│   └── SnakeGame.java
└── README.md
```

## How to Run the Game

1. Ensure you have Java Development Kit (JDK) installed on your machine.
2. Open a terminal and navigate to the `snake-game` directory.
3. Compile the Java file using the following command:
   ```
   javac src/SnakeGame.java
   ```
4. Run the game with the command:
   ```
   java -cp src SnakeGame
   ```

## Additional Information

- The game window is set to 600x600 pixels and titled "Snake".
